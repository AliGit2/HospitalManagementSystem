## 📌 Overview

This Hospital Management System is a comprehensive web application designed to streamline hospital operations by managing:
- **Patient Records** - Complete patient information and medical history
- **Doctor Management** - Physician information and schedules
- **Ward Management** - Hospital ward organization and details
- **Bed Allocation** - Bed availability and patient assignments
- **Admissions** - Patient admission tracking and processing

## ✨ Features

- 🏥 **Patient Management** - Create, update, and manage patient records
- 👨‍⚕️ **Doctor Management** - Manage doctor profiles and specializations
- 🏢 **Ward Management** - Organize and track hospital wards
- 🛏️ **Bed Management** - Monitor bed availability and allocations
- 📝 **Admission Tracking** - Track and manage patient admissions
- 🔍 **Search & Filter** - Quickly find patients, doctors, and other records
- 📊 **Dashboard** - Overview of hospital operations
